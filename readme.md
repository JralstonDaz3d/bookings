# Bookings and Reservations

This is the repo for my Bookings and Reservations application.

- Built using Go version 1.17
- Uses the following external packages
  - [SCS Session Management v2.5.0](https://github.com/alexedwards/scs/v2 v2.5.0) 
  - [Chi Router v5.0.7](https://github.com/go-chi/chi/v5 v5.0.7)
  - [NoSurf CSRF v1.1.1](https://github.com/justinas/nosurf v1.1.1)
